# Анализ рынка зарплат

Одностраничный сайт для быстрого расчета зарплатных ожиданий кандидатов по данным из HR-аналитики.

Страница помогает заполнить основные параметры вакансии, внести количество кандидатов по зарплатным сегментам и получить готовый блок отчета: количество целевых кандидатов, количество кандидатов с указанной зарплатой, медиану ожиданий и распределение по диапазонам.

## Что внутри

- `index.html` - разметка страницы.
- `styles.css` - визуальное оформление, цвета, адаптивность.
- `script.js` - расчеты, сворачивание блоков, копирование и выгрузка таблицы.

## Как пользоваться

1. Откройте `index.html` в браузере.
2. Заполните блок `Основные данные`.
3. Внесите количество кандидатов в зарплатных сегментах.
4. Справа автоматически обновится отчет `Зарплатные ожидания кандидатов`.
5. При необходимости скройте блок ввода, чтобы оставить только аккуратный отчет.
6. Скопируйте таблицу или скачайте данные в формате TSV через кнопки в блоке действий.

## Как считается медиана

Так как вводятся не точные зарплаты каждого кандидата, а количество кандидатов по диапазонам, медиана считается приблизительно по сгруппированным данным.

Логика такая:

1. Скрипт считает общее количество кандидатов, указавших зарплату.
2. Находит позицию 50% выборки.
3. Определяет зарплатный диапазон, в который попадает эта позиция.
4. Оценивает медиану внутри этого диапазона пропорционально накопленному количеству кандидатов.

## Цветовая подсветка

В таблице `Зарплатные ожидания в разбивке по суммам` столбец с долей сегмента подсвечивается зеленым относительно самого крупного сегмента в текущей таблице.

Самый большой сегмент получает самый насыщенный зеленый оттенок, меньшие сегменты становятся светлее, а нулевые значения остаются белыми.

## Публикация на GitHub Pages

1. Создайте репозиторий на GitHub.
2. Загрузите в корень репозитория файлы:
   - `index.html`
   - `styles.css`
   - `script.js`
   - `README.md`
3. Откройте настройки репозитория.
4. Перейдите в `Pages`.
5. В разделе `Build and deployment` выберите ветку, например `main`, и папку `/root`.
6. Сохраните настройки.

Через некоторое время сайт будет доступен по ссылке GitHub Pages.

## Ограничения

- Сайт не загружает данные с hh.ru автоматически.
- Все значения вводятся вручную.
- Расчет медианы является оценочным, потому что исходные данные сгруппированы по диапазонам.
